# Typora快捷键

| -------- | ---------------- |
| 标题     | CTRL + 1         |
| 添加引用 | CTRL + SHIFT + Q |
| 添加表格 | CTRL + T         |
| 添加代码 | CTRL + SHIFT + k |
| 查看目录 | CTRL + SHIFT + L |
| 加粗文字 | CTRAL + B        |
